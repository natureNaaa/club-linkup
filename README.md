# club-linkup
LinkUp - 大学生社团招新智能匹配平台 🚀

项目定位：通过 AI 兴趣画像与智能标签算法，解决高校社团招新中的信息不对称问题。

✨ 核心亮点

简洁高级感设计：采用现代 UI 规范，具备流体布局与毛玻璃交互特效。

智能测评引擎：5 维兴趣画像分析，自动计算用户与社团的契合度。

一键联动申请：打通个人简历与社团投递，实现极致的报名体验。

响应式架构：适配移动端与 PC 端，随时随地开启招新。

🛠️ 技术栈

Frontend: React.js + Tailwind CSS

Icons: Lucide React

Deployment: Vercel (CI/CD 自动化部署)

Data Architecture: 基于 LocalStorage 的轻量化状态持久化

📖 如何快速开始

访问 在线预览地址

进入“智能匹配”开始 1 分钟趣味测评。

在“社团广场”探索感兴趣的组织并完成一键申请。

本项目为测评题目演示版本，重点展示产品结构与核心功能逻辑。
